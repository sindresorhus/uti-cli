# uti-cli

> Get the [UTI](https://en.wikipedia.org/wiki/Uniform_Type_Identifier) (Uniform Type Identifier) of a file on macOS


## Install

```
$ npm install --global uti-cli
```


## Usage

```
$ uti --help

  Example
    $ uti unicorn.md
    net.daringfireball.markdown
```
